Exemplos de semantica

Negrito

tenho aqui a palavra termo apenas em negrito usando b sendo apenas como forma - sendo um exemplo de forma não semantica

Já aqui a palavra termo está em negrito porquê é uma palavra forte utilizando a teg strong sendo seu significado - sendo um exemplo de forma semantica

Italico

tenho aqui a palavra termo apenas em Italico usando i sendo apenas como forma - sendo um exemplo de forma não semantica

Já aqui a palavra termo está em Italico porquê é uma palavra com enfase utilizando a teg em sendo seu significado - sendo um exemplo de forma semantica

Texto marcado

tenho aqui a palavra termo para um texto marcado usando a teg mark

Testo grande e texto pequeno

Aqui está um texto grande usando a teg big e um texto pequeno usando a teg small

Texto excluido

um texto escluido é um texto que pode ser lido mas não considerado usando a teg del

texto incerido

existe o texto incerido para dar uma enfase e indicar que tem que ser lido com atenção usando a teg ins - sendo um significado, logo um semantica

existe tambem o texto incerido com a teg u para apenas sublinhar o texto - porem isso é uma forma - logo não semantico

Texto sobre escrito

par incerir x20 + 3 usando a teg sup

Texto sob escrito

para inseir h2o usando a teg sub
